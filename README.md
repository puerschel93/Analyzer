### WebScraper
A script using Puppeteer to collect .scss, .sass, .less and .styl data from github.
This is script is used exclusively for my bachelorthesis.

-----

Analyzable files are not included.
Add files to `src/input/:extension`.

-----

### How to use


To run the analyzer
```
npm run analyze
```

To run the decloner
```
npm run declone
```
